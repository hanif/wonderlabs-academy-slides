Philosophy


The Unix philosophy emphasizes building simple, short, clear, modular, and extensible code that can be easily maintained and repurposed by developers other than its creators. The Unix philosophy favors composability as opposed to monolithic design.

Philosophy

  • Everything is a File (Device I/O, Network I/O,O S Runtime, etc)
  • Do one thing and do it well (ls, less, more, cat)
  • Write programs to work together
  • Make every program a filter

Secure Shell (SSH)

Secure Shell (SSH) is a cryptographic network protocol for operating network services securely over an unsecured network. The best known example application is for remote login to computer systems by users.

Secure Shell (SSH)

  • It's a protocol
  • Use ssh to access remote shell
  • Difference between ssh and telnet
  • Can be authenticated using identity and credentials or public key
  • Be careful! even though the connection is encrypted, but your command can be logged
  • SSH protocol also used in file transfer mechanism: scp, rsync, sftp

Secure Shell (SSH)

  • ssh user@host
  • ssh -D 2121 user@host
  • scp ~/Documents/hello.txt user@host:~/my-private-folder
  • scp user@host:~/my-file.pdf ~/Downloads
  • Not to be confused with ssh, scp, and sftp
